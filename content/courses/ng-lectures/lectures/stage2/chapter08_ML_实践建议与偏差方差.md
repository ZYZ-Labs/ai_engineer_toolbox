# 第8章 ML 实践建议与偏差方差

## 目标

建立诊断模型的系统方法。

---

## 必学内容

### 训练集 / 验证集 / 测试集

```
训练集：学参数
验证集：调超参
测试集：最终评估
```

### 偏差 vs 方差诊断

```
高训练误差 + 高验证误差 → 高 bias（欠拟合）
低训练误差 + 高验证误差 → 高 variance（过拟合）
```

### 解决策略

高 bias：

-  bigger network
-  train longer
-  换更复杂模型

高 variance：

-  more data
-  regularization
-  简化模型
