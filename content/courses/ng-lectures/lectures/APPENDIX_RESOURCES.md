# 课程速查表

| 阶段 | 来源 | 章节 | 核心主题 |
|------|------|------|----------|
| 一 | ML Spec C1 | 1-4 | 回归、分类、梯度下降、正则化 |
| 二 | ML Spec C2 | 5-9 | 神经网络、TensorFlow、决策树 |
| 三 | DL Spec C1-C2 | 10-14 | 深层网络、优化、BatchNorm |
| 四 | DL Spec C3-C4 | 15-19 | ML策略、CNN、ResNet、检测 |
| 五 | DL Spec C5 | 20-22 | RNN、词嵌入、Attention |
| 六 | ML Spec C3 + GenAI | 23-28 | 无监督学习、RL、LLM、Agent |

---

> 学习顺序建议：按阶段顺序进行，不要跳阶段。
>
> 每个阶段结束后，至少完成一个对应的小项目再进入下一阶段。
