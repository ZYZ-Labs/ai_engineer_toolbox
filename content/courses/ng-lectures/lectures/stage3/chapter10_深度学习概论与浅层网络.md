## 必学内容

### 为什么深度学习兴起

```
数据量 ↑ + 计算力 ↑ + 算法改进 → 深度网络优势显现
```

### 浅层神经网络

```python
Z[1] = W[1]X + b[1]
A[1] = g(Z[1])
Z[2] = W[2]A[1] + b[2]
A[2] = g(Z[2])
```

### 向量化实现

```python
# m 个样本同时计算
Z = np.dot(W, X) + b
A = g(Z)
```

### 计算图

```python
X → Linear → ReLU → Linear → Sigmoid → L
```
