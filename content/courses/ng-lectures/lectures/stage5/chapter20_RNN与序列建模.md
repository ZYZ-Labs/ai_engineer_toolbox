## 必须真正理解

### 为什么标准RNN有梯度消失

```
W_aa 连乘多次 → 指数级放大/缩小
```

### LSTM 如何解决

```
记忆单元 c<t> 有加法路径
梯度可以直接流过，不受乘法影响
```
